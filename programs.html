We offer many programs at CT STEM KIDS CLUB. The programs are as follows.

  • M-ELMENTARY1- A math class for Kids in grades K-2 teaching a intro to numbers operation and really basic inequalities. The class also teaches addition and subtraction. We offer a friendly environment to younger kids. It is a fun class.

  • M-ELMENTARY2- A course involving use of the Beast Academy Series from Art of Problem Solving. It teaches more advanced topics than the previous, such as Geometry, Multiplication. This course is for kids in third or fourth grade.

  • M-Prealgebra- A course involving use of the Prealgebra Book from AOPS. The kids who do this class will learn topics like one step equations, MEAN/MEDIAN/MODE, and age equations. It is a fun and easy course, and will offer your child a brief intro to mastering Algebra.

  • M-Algebra- A course using the books, Algebra, Geometry, Number Theory, and Probability Books from AOPS. Kids learn formulas, and equation solving algorithms that make math fun and easy.

  • S-Middle School- A course preparing students for the National Science Bowl for middle scholars. It teaches topics in Life Science, Earth Science, Math, General Science, Chemistry, and Physics. The class alternates science field every week and each field has a different teacher.

  • T-HTMLJavaScript-B- A intro to coding webpages for beginners. It teaches HTML and basic Java Script. They learn tags to make paragraphs, alerts, tables and more.

  • T-HTMLJavaScript-A- An Advanced coding group, who uses Advanced HTML and Java Script. They use the book HTML, CSS and JavaScript All in One, Sams Teach Yourself...

  • E-Robotics- A course showing who to design and code a robot.

  • Vocabulary-Elementary- A course following Level Red of the Sadlier Vocabulary Workshop.Sadlier Vocabulary Workshop is a company that makes textbooks for kids to learn vocabulary. The beginner level books range from Red to Blue, and more advanced lvels go from A to H. The book for Level Red uses some words that are begginer for little kids, and others which are more advanced. The words are taught through different picture stories of different genres. There is also a set of Common Core words taught as part of our curriculum. These words are important words from STEM, the arts, and sports (even LA can fit into STEM!), and some are words used in Standardized testing for Kindergartners, 1st Graders, and 2nd Graders.
